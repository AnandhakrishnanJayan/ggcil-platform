# GGCIL Platform

Green Gold Carbon Intelligence Ltd – Sustainability Intelligence Platform.

This platform is designed to help organizations measure, analyze, and reduce carbon emissions through AI-powered insights.

## Core Features
- Carbon audit calculations
- Sustainability analytics dashboard
- AI-powered energy optimization
- Digital twin simulation engine
- Carbon credit tracking

## Technology Stack
Backend: Python FastAPI  
AI Engine: Scikit-learn  
Frontend: React / Next.js  
Database: PostgreSQL
